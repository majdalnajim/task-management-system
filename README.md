# Task Management System 
A full-featured Task Managment System built using Laravel.
This application allows users to manage tasks through a clean web interface with authentication, task status tracking, and pagination.
## Features 
- User Authentication (Login/Register)
- Ctreate,Read,Update,Delete Tasks
- User-specific tasks(eash user sees only their tasks)
- Task status (completed/pending)
- Search functionality
- Ragination for task listing
## Technplogies
- Laravel
- php
- MySQL
- Blade (laravel templating)
## Project Structure
- Routes-> Handle requests
- Controllers-> Business logic
- Models-> Database interaction
- view -> Blade tamplates
## How to run
1. Clone the repository
2. Run composer install
3. Create .env file
4. Run php artisan migrate
5. Run php artisen serve
## Usage
Open in browser: http://127.0.0.1:8000
After registering/logging in,users can:
. Manage their tasks
. Update task status
. Navigate through paginated tasks
## Purpose
This project was built to practice Laravel fundamentals and develpo a real-world web application with authentication and user interaction.
